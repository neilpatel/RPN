# My Custom RPN+ Calculator

This project is an interactive Reverse Polish Notation (RPN+) calculator written in Ruby 2.5.0.

## Usage
+ Clone the repository
	+ `git clone https://github.com/neilpatel/RPN.git`
+ To Execute RPN:
	+ 'ruby rpn.rb'
